# Runbook: Data Subject Request

**Severity:** Medium
**Estimated Time:** 30-90 minutes
**Requires:** Admin access, compliance approval

## Symptoms
- Data access or deletion request received
- Legal requirement to respond within timeline

## Diagnosis
1. Verify requester identity and scope.
2. Identify data sources and retention constraints.

## Resolution Steps
1. Export or delete data as required by policy.
2. Document all actions and approvals.
3. Provide confirmation to requester.

## Verification
- [ ] Data export or deletion completed
- [ ] Audit log updated

## Post-Incident
- [ ] Record request outcome
- [ ] Review any delays or blockers

## Escalation
If request cannot be fulfilled, escalate to legal counsel.
