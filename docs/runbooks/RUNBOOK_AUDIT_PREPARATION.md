# Runbook: Audit Preparation

**Severity:** Medium
**Estimated Time:** 1-3 hours
**Requires:** Admin access, compliance owner

## Symptoms
- External audit scheduled
- Evidence bundle request received

## Diagnosis
1. Identify audit scope and time period.
2. Confirm required evidence and controls.

## Resolution Steps
1. Generate evidence bundle and control matrix.
2. Export relevant audit logs.
3. Review for completeness and redactions.

## Verification
- [ ] Evidence bundle generated successfully
- [ ] Audit logs cover required timeframe

## Post-Incident
- [ ] Store audit artifacts securely
- [ ] Record auditor questions and responses

## Escalation
If required evidence is missing, escalate to compliance lead.
