# External Data Integration - User Guide

**Version:** 1.0.0

---

## Overview

The External Data Integration module enriches your CBAs with real-world labor market data from Statistics Canada, Provincial Labour Relations Boards, and the Canadian Labour Congress.

---

## Accessing External Data

### 1. View Wage Benchmarks

Navigate to any CBA and click the **"Benchmark"** tab to see:

- **Market Wages**: Compare your wages against industry averages
- **Wage Trends**: Historical wage growth in your sector
- **Regional Comparisons**: How your wages compare by province

### 2. Search LRB Agreements

Use the **Agreement Search** to find comparable agreements:

1. Go to **CBA Intelligence** → **Search Agreements**
2. Filter by:
   - **Employer Name**
   - **Union**
   - **Jurisdiction** (ON, BC, etc.)
   - **Sector** (Healthcare, Education, etc.)
3. Click **Search** to see results

### 3. Union Density Reports

View union density statistics for your sector:

1. Go to **Analytics** → **Union Density**
2. Select:
   - **Sector**: Your industry
   - **Geography**: Province or national
   - **Time Period**: Year-over-year comparison

## Understanding the Data

### Wage Benchmarks

| Field | Description |
|-------|-------------|
| **Median Wage** | 50th percentile wage in your occupation |
| **P10** | 10th percentile (entry-level) |
| **P90** | 90th percentile (senior level) |
| **Source** | Statistics Canada, Labour Force Survey |
| **Last Updated** | Most recent data refresh |

### Union Density

| Field | Description |
|-------|-------------|
| **Density %** | Percentage of workers unionized |
| **Coverage %** | Workers covered by union contract |
| **YoY Change** | Year-over-year change |

### LRB Agreements

| Field | Description |
|-------|-------------|
| **Employer** | Company name |
| **Union** | Union name and local |
| **Effective Date** | When agreement started |
| **Expiry Date** | When agreement ends |
| **Sector** | Industry classification |

## Common Use Cases

### 1. Prepare for Bargaining

1. **Review Market Wages**
   - Go to **CBA Intelligence** → **Benchmarks**
   - Select your NOC code and province
   - Note the median wage and trends

2. **Find Comparable Agreements**
   - Search LRB for similar employers
   - Note wage progression schedules
   - Review benefit packages

3. **Analyze Trends**
   - Check inflation rates
   - Review recent settlement trends
   - Identify patternsettlements

### 2. Benchmark Your CBA

1. Open your CBA
2. Click **"Compare to Market"**
3. View:
   - Wage gap analysis
   - Benefit comparison
   - Recommendations

### 3. Research Arbitration Decisions

1. Go to **Precedents** → **Search**
2. Filter by:
   - Issue type
   - Tribunal
   - Time period
3. Review similar cases

## Data Sources

### Statistics Canada

- **What**: Official government labor market data
- **Update Frequency**: Monthly
- **Coverage**: All provinces, all industries
- **Source**: https://www.statcan.gc.ca

### Provincial Labour Relations Boards

- **Ontario LRB**: ~5,000 active agreements
- **BC LRB**: ~3,000 active agreements
- **Update Frequency**: Weekly
- **Source**: https://www.olrb.gov.on.ca | https://www.lrb.bc.ca

### Canadian Labour Congress

- **What**: Per-capita and bargaining benchmarks
- **Update Frequency**: Quarterly
- **Access**: Partnership required

## FAQs

### How current is the data?

- **Wage Data**: Updated monthly (1-2 months behind)
- **LRB Agreements**: Updated weekly
- **Union Density**: Updated annually

### Can I export data?

Yes! Click the **"Export"** button on any report to download:
- CSV format
- Excel format
- PDF format

### How accurate is the data?

Statistics Canada data is official government statistics with high accuracy. LRB data comes directly from filed agreements. Always verify with primary sources for legal purposes.

### What if data is missing?

Some niche occupations or regions may have limited data. Contact support if you need assistance finding comparable data.

## Troubleshooting

### "No benchmark data available"

- Try broadening your search criteria
- Check if NOC code is correct
- Contact support for custom research

### "Data appears outdated"

- Clear your browser cache
- Check the "Last Updated" timestamp
- Contact support if issue persists

### "Can't find specific agreement"

- Try partial employer name searches
- Check alternate spellings
- Search by union instead
